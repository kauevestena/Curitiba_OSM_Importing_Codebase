# Curitiba_OSM_Importing_Codebase
This repository holds the code for the data importing from the "Geocuritiba" Portal, available at https://geocuritiba.ippuc.org.br/portal/apps/sites/#/geocuritiba, to OpenStreetMap

# Reproducing:

- Clone
- Make a virtual environment
- Install dependencies using "requirements.txt"
- The "scripts" folder has some ready-to-go stuff
